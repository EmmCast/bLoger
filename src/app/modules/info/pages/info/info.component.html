Información del proyecto

Objetivo del proyecto

El objetivo de BLoger es proporcionar un espacio donde los usuarios puedan compartir y consultar recomendaciones de entretenimiento, permitiendo almacenar información de manera local, visualizar detalles, agregar comentarios y organizar el contenido por categorías.

Este proyecto fue desarrollado como evidencia práctica de los conceptos aprendidos durante la asignatura de Desarrollo de Aplicaciones en Red, aplicando componentes reutilizables, enrutamiento, formularios, almacenamiento local y diseño responsivo con Angular.

BLoger es un blog estudiantil desarrollado para la materia de Desarrollo de Aplicaciones en Red. La aplicación permite al usuario interactuar con diferentes recomendaciones de películas, series, anime, libros y videojuegos.

¿Qué puedes hacer en BLoger?

  • Registrarte de forma local con un nombre de usuario.
  • Consultar recomendaciones por categoría.
  • Agregar nuevas películas, series, anime, libros o videojuegos.
  • Filtrar contenido por género y año.
  • Consultar detalles de cada recomendación.
  • Agregar comentarios en las recomendaciones.

Guía rápida de uso

1. Iniciar sesión

Ingresa un nombre de usuario para acceder a la aplicación.

2. Elegir una categoría

Selecciona si deseas ver películas, series, anime, libros o videojuegos.

3. Consultar recomendaciones

Explora el catálogo y utiliza los filtros para encontrar contenido por género o año.

4. Agregar una recomendación

Llena el formulario con título, género, año, descripción e imagen.

5. Comentar

Dentro del detalle de cada recomendación puedes dejar comentarios.

Tecnologías utilizadas

La aplicación fue desarrollada con Angular, TypeScript, HTML y CSS. La información se guarda de manera local usando LocalStorage, por lo que no requiere base de datos.
